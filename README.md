# TaskFlow Lite

## 📌 Description
TaskFlow Lite is a responsive task management application developed using HTML, CSS, and Vanilla JavaScript.

The application allows users to:
- Add tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks
- Store data using localStorage

---

## 🚀 Features

- CRUD Operations
- Responsive Design
- Filter Controls
- Form Validation
- localStorage Support
- Task Statistics

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
